# loyalty-web-backoffice

Frontend backoffice de la loyalty platform.

## Propósito
Concentrar operación interna y visibilidad transversal:
- soporte y auditoría
- revisión de clientes
- revisión de órdenes
- monitoreo operativo

## Dependencias objetivo
- `loyalty-bff-backoffice`

## Estado
Bootstrap funcional con dashboard operativo, detalle de cliente y detalle de orden usando BFF backoffice con fallback mock.
